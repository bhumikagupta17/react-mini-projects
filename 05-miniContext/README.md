# 🔐 Login App with React Context

A simple React mini project demonstrating **state management using Context API**.  
This app implements a basic login system with global authentication state and protected UI.

---

## 🚀 Features

- 🔑 User Login / Logout
- 🌐 Global state management using React Context API
- 🔒 Basic authentication flow
- 👤 Displays logged-in user info
- ⚡ Built with Vite for fast performance

---

## 🛠️ Tech Stack

- React
- Vite
- Context API
- JavaScript (ES6+)
- CSS

---
## 🧠 What I Learned

- How to use **React Context API** for global state  
- Managing authentication state across components  
- Structuring a scalable React project  
- Handling simple login logic without backend
