# React Mini-Projects ⚛️

A collection of small React applications built while learning frontend development. This repository serves as a timeline of my progress with React hooks, component architecture, and UI design.

## 🚀 Projects

| # | Project Name | Description | Concepts Practiced |
|---|---|---|---|
| 01 | [Background Changer](./01-bgChanger) | A dynamic UI that updates screen colors. | `useState`, Tailwind CSS, Event Handling |
| 02 | [Password Generator](./02-passwordGenerator) | A customizable random password generator with a copy-to-clipboard feature. | `useState`, `useEffect`, `useCallback`, `useRef` |
| 03 | [Currency Converter](./03-currencyConvertor) | A real-time currency converter using live API data and reusable input components. | Custom Hooks, API Fetching, `useEffect`, `useState` |

## 🛠️ Tech Stack
* React (Vite)
* JavaScript (ES6+)
* Tailwind CSS

## 💻 How to Run Locally

Since this repository is a collection of individual React applications, you will need to navigate into the specific project folder you want to view before starting the server.

**1. Clone this repository to your local machine:**
```bash
git clone [https://github.com/your-username/react-mini-projects.git](https://github.com/your-username/react-mini-projects.git)
```

**2. Navigate into the directory of the project you want to run (e.g., Currency Converter):**
```bash
cd 03-currencyConvertor
```

**3. Install the necessary dependencies:**
```bash
npm install
```

**4. Start the Vite development server:**
```bash
npm run dev
```
**Dont forget to swap out your ```username``` in the clone link before you commit!!**
