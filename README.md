# React Mini-Projects ⚛️

A collection of small React applications built while learning frontend development. This repository serves as a timeline of my progress with React hooks, component architecture, and UI design.

## 🚀 Projects

| # | Project Name | Description | Concepts Practiced |
|---|---|---|---|
| 01 | [Background Changer](./01-bgChanger) | A dynamic UI that updates screen colors. | `useState`, Tailwind CSS, Event Handling |
| 02 | [Password Generator](./02-passwordGenerator) | A customizable random password generator with a copy-to-clipboard feature. | `useState`, `useEffect`, `useCallback`, `useRef` |
| 03 | [Currency Converter](./03-currencyConvertor) | A real-time currency converter using live API data and reusable input components. | Custom Hooks, API Fetching, `useEffect`, `useState` |
| 04 | [React Router App](./04-reactrouter) | A multi-page responsive website with a persistent layout and optimized API data fetching. | `react-router-dom`, `useLoaderData`, Nested Routes, Layouts |
| 05 | [Login with Context](./05-context-login) | A basic login system with global authentication state using Context API. | Context API, State Management, Conditional Rendering |
| 06 | [Theme Toggle Product Card](./06-theme-toggle) | A product card UI with global light/dark mode toggle. | Context API, Theming, Dynamic Styling |
| 07 | [Todo App](./07-todolist) | A full CRUD todo app with localStorage persistence and dynamic UI updates. | CRUD, localStorage, State Management, Conditional Rendering |

## 🌱 Learning Journey

This repository documents my journey of learning React step by step:

- Started with basic state management using `useState`
- Learned side effects and optimization with `useEffect`, `useCallback`, and `useRef`
- Built reusable components and improved component architecture
- Explored routing using `react-router-dom`
- Worked with APIs and custom hooks
- Learned global state management using Context API
- Implemented features like authentication and theme toggling
- Built CRUD-based applications with dynamic UI updates
- Used `localStorage` to persist user data across sessions
- Improved styling skills using Tailwind CSS and responsive layouts

## 💻 How to Run Locally

Since this repository is a collection of individual React applications, you will need to navigate into the specific project folder you want to view before starting the server.

**1. Clone this repository to your local machine:**
```bash
git clone [https://github.com/your-username/react-mini-projects.git](https://github.com/your-username/react-mini-projects.git)
```

**2. Navigate into the directory of the project you want to run (e.g., Currency Converter):**
```bash
cd 03-currencyConvertor
```

**3. Install the necessary dependencies:**
```bash
npm install
```

**4. Start the Vite development server:**
```bash
npm run dev
```
**Dont forget to swap out your ```username``` in the clone link before you commit!!**
