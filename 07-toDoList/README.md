# ✅ Todo App with Local Storage (React)

A simple and interactive Todo application built with React that supports **adding, editing, deleting, and completing tasks**, with data persistence using **localStorage**.

---

## 🚀 Features

- ➕ Add new todos  
- ✏️ Edit existing todos  
- ❌ Delete todos  
- ✅ Mark todos as complete  
- 🎨 Dynamic styling for completed tasks  
- 💾 Data persistence using localStorage  
- ⚡ Fast and responsive UI  

---

## 🛠️ Tech Stack

- React  
- Vite  
- JavaScript (ES6+)  
- CSS  
- localStorage API  

---

## 🧠 What I Learned

- Managing state in React for dynamic lists  
- Implementing CRUD operations (Create, Read, Update, Delete)  
- Using **localStorage** to persist data  
- Conditional rendering and dynamic styling  
- Handling user interactions efficiently
