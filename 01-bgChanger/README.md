# Background Color Changer 🎨

A lightweight React application that changes the background color of the screen dynamically based on user selection.

## 🧠 What I Learned

Building this project helped solidify the following React and frontend concepts:
* **The `useState` Hook:** Learned how to store, initialize, and update the `color` state dynamically.
* **Inline Styling:** Practiced passing React state directly to the `style` attribute (`style={{backgroundColor: color}}`).
* **Tailwind CSS:** Utilized utility classes for rapid layout building (e.g., `flex`, `justify-center`, `rounded-2xl`, `shadow-xl`).
* **Event Handling with Callbacks:** Used inline arrow functions in the `onClick` handler (`onClick={() => setColor("red")}`) to prevent the function from firing on page load.

## 💻 Run Locally

To run this specific project on your machine:

1. Clone the repository.
2. Navigate to this folder: `cd react-mini-projects/01-bgChanger`
3. Install dependencies: `npm install`
4. Start the dev server: `npm run dev`
