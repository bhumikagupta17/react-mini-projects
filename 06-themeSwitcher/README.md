# 🎨 Theme Toggle Product Card (React)

A simple React mini project demonstrating **light/dark mode toggle using Context API** with a clean **product card UI**.

---

## 🚀 Features

- 🌗 Light / Dark mode toggle
- 🌐 Global theme management using React Context API
- 🛍️ Responsive product card UI
- ⭐ Product rating display
- 🛒 "Add to Cart" button (UI only)
- ⚡ Fast performance with Vite

---

## 🛠️ Tech Stack

- React
- Vite
- Context API
- JavaScript (ES6+)
- CSS

---

## 🧠 What I Learned

- How to use **React Context API** for theme management  
- Implementing **dark/light mode toggle globally**  
- Building reusable UI components  
- Styling based on dynamic state
